## 📌 Project Overview

**Beautiful Portfolio** is a single-page personal website built with **React 19** and **Tailwind CSS 4**, designed to present my background, technical skills, and selected projects in a clean, responsive layout.  
The site features smooth section-based navigation, an animated **star background**, **light/dark theme switching**, and a working **contact form** powered by EmailJS.  

It is structured around modular sections — Hero, About, Skills, Projects, and Contact — and built with reusable, accessible UI primitives from Radix UI, polished with Lucide and React Icons for a consistent visual style.

---

## 🚀 Features

- **🏠 Hero Section** – Animated intro with name, role, and a call-to-action.
- **👤 About Section** – Personal bio, background, and highlights.
- **🛠 Skills Section** – Categorized showcase of my tech stack.
- **💼 Projects Section** – Featured projects with descriptions, tech badges, and live/GitHub links.
- **📬 Contact Section** – Working contact form integrated with **EmailJS** (no backend required).
- **🌗 Theme Toggle** – Seamless switching between light and dark modes.
- **✨ Animated Star Background** – Subtle animated stars for a modern visual feel.
- **🧭 Navbar & Footer** – Smooth in-page navigation and consistent footer across sections.
- **🔔 Toast Notifications** – Accessible feedback messages via Radix UI Toast.
- **🚦 Custom 404 Page** – Branded "Not Found" route for unknown URLs.
- **📱 Fully Responsive** – Mobile-first design, optimized for all screen sizes.
